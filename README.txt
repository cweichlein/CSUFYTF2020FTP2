STEPS FOR FTP COMMUNICATION

1.) Run server. Server will print it's ID before the "Listening for clientele" message.
2.) Run Client (inside inbox directory). Client will ask for the server's ID
3.) Enter the ID and hit enter on keyboard.
4.) On server, enter the name of the file (and its extension) to be sent. Hit enter.
5.) On client, enter the name of the file (and its extension) to be received. Hit enter.
6.) The file should now be in the archive containing the client.py file. Voila!
